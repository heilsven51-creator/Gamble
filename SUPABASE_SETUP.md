# Supabase Setup

1. Create a Supabase project.
2. Open the SQL editor and run [supabase-setup.sql](/C:/Users/susan/Documents/Gamble/supabase-setup.sql).
3. Copy [supabase-config.example.js](/C:/Users/susan/Documents/Gamble/supabase-config.example.js) to `supabase-config.js`.
4. Fill in your project URL and anon key in `supabase-config.js`.
5. After that, the frontend can be wired to the live Supabase project.

You still need to share these two values if you want me to finish the full live integration:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
